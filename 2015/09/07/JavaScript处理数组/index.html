JavaScript处理数组

发表于 2015-09-07 | 分类于 JavaScript | | 阅读次数:

JavaScript只支持一维数组，但其作为一门弱类型语言，数据类型相对灵活，如果使保存在数组中的元素是数组的话，就可以轻松创建多维数组。

二维数组

在C语言中我们已经熟悉二维数组的概念，二维数组类似一种由行和列构成的数据表格。
